# LuminaBook — AI-First Scheduling, Built End-to-End with AI

An intelligent scheduling platform that goes far beyond traditional tools like Calendly.

Powered by advanced AI, LuminaBook lets users book meetings using natural language, automatically finds the best time slots across multiple calendars, resolves conflicts intelligently, sends smart reminders, and even drafts personalized follow-up messages.

This entire project was developed from scratch using AI assistance (planning → architecture → coding → testing → documentation), showcasing modern full-stack development with AI pair-programming.

## Key Features

- Natural language event creation ("Book a 45-min strategy call with John next week")
- AI-powered smart availability & conflict resolution
- Multi-calendar sync (Google, Outlook, etc.)
- Automated reminders and thank-you notes
- Personalized meeting suggestions
- Beautiful, responsive booking pages
- Full admin dashboard

Built as a demonstration of end-to-end AI-assisted software development.

Tech stack: JHipster Java (backend), Spring Boot, TypeScript, Angular (fronend), PostgreSQL
